I'm Dira Halimatu Zahra

Selamat datang di portofolio pribadi saya. Saya adalah seorang desainer dan pengembang yang penuh semangat, dengan keahlian dalam menciptakan situs web dan aplikasi yang indah serta fungsional.

Skill saya berfokus pada bidang desain, khususnya dalam menciptakan tampilan visual yang menarik dan mudah dipahami oleh pengguna. Dalam proses desain, saya terbiasa menggunakan dua alat utama, yaitu Figma dan Canva. Figma saya gunakan untuk kebutuhan desain antarmuka (UI/UX) yang lebih kompleks, seperti wireframe, prototype aplikasi, atau website, karena fitur kolaborasinya yang kuat dan fleksibel. Sementara itu, Canva sering saya gunakan untuk membuat desain grafis yang lebih ringan dan cepat, seperti konten media sosial, poster, dan presentasi.

Figma

Saya memiliki keterampilan dalam menggunakan Figma untuk membuat desain antarmuka (UI) website yang responsif dan estetis. Saya terbiasa membuat wireframe, layout halaman, serta prototype interaktif yang dapat digunakan untuk pengujian dan presentasi kepada klien atau tim pengembang. Selain itu, saya mampu menerapkan prinsip desain UI/UX, konsistensi elemen visual, dan struktur informasi yang baik untuk menciptakan pengalaman pengguna yang optimal.

VS CODE

Saya terbiasa menggunakan Visual Studio Code (VS Code) sebagai text editor utama dalam mengembangkan proyek front-end maupun back-end. Saya memanfaatkan berbagai ekstensi seperti Live Server, Prettier, Emmet, dan GitLens untuk mempercepat workflow dan menjaga kualitas kode. Saya juga mampu mengatur struktur folder proyek, menggunakan terminal bawaan, serta melakukan integrasi dengan Git untuk version control
